### How to install.
```bash
git clone https://github.com/hurt-6666/Discord-bot
cd Discord-bot
npm init
npm install
node index
```
### Things you need!
You will 100% need mongoDB and knowlage of how to use it that can be found here : https://docs.mongodb.com/v3.2/tutorial/install-mongodb-on-windows/
